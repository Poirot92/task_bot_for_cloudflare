# 🤖 Telegram Task Bot — Cloudflare Workers

Полнофункциональный таск-менеджер для команд в Telegram. Без сервера, без затрат — работает на Cloudflare Workers + D1.

---

## ✨ Что умеет бот

### Для руководителя
- Создание организации с уникальным кодом приглашения
- Назначение задач сотрудникам с дедлайном через встроенный календарь (08:00–22:00)
- **Изменение дедлайна** задачи в любой момент — сотрудник получает уведомление
- Назначение встреч с выбором участников и места
- Просмотр всех задач команды и просроченных
- Одобрение / отклонение заявок на вступление
- Управление составом команды
- Календарь на неделю (задачи + встречи)

### Для сотрудника
- Вступление в организацию по коду
- Просмотр своих задач и статусов
- Смена статуса: `Ожидает → В работе → Завершена`
- Комментирование задач
- Просмотр предстоящих встреч

## 🔔 Умная система напоминаний

Бот сам определяет когда и кому напомнить — в зависимости от длительности задачи.

### Задача не взята в работу
| Прошло с момента создания | Кому | Тон |
|---|---|---|
| 2 часа | Руководителю | С лёгким юмором 😄 |
| 24 часа | Руководителю | Серьёзно 🚨 |

### Напоминания до дедлайна
| Длительность задачи | Сотруднику | Руководителю |
|---|---|---|
| ≥ 3 дней | За 24 часа | За 3 часа |
| 1–3 дня | За 6 часов | За 2 часа |
| < 24 часов | За 3 часа + за 1 час | За 1 час |

### После просрочки
| Время | Кому |
|---|---|
| Сразу | Сотруднику — шанс исправить |
| +1 час тишины | Руководителю |
| Каждые сутки | Руководителю — пока не закроют |

### Встречи
- За 1 час — всем участникам
- За 30 минут — руководителю

## 🏗 Архитектура

```
Telegram (webhook)
       ↓
Cloudflare Workers (JS)
       ↓
Cloudflare D1 (SQLite)
       ↑
Cron Triggers (каждые 15 минут) → напоминания
```

| Компонент | Решение |
|---|---|
| Язык | JavaScript (ES2022) |
| Хостинг | Cloudflare Workers |
| База данных | Cloudflare D1 (SQLite) |
| Получение апдейтов | Webhooks |
| Напоминания | Cron Triggers |
| Состояния диалогов | State Management через D1 |

## 🗄 База данных

| Таблица | Назначение |
|---|---|
| `organizations` | Организации и коды приглашений |
| `users` | Пользователи (boss / employee) |
| `tasks` | Задачи с дедлайнами и статусами |
| `comments` | Комментарии к задачам |
| `meetings` | Встречи |
| `meeting_participants` | Участники встреч |
| `join_requests` | Заявки на вступление |
| `reminders_sent` | Отправленные напоминания |
| `user_states` | Состояния диалогов (ConversationHandler) |

## 🚀 Деплой

```bash
# 1. Установи Wrangler
npm install -g wrangler
wrangler login

# 2. Создай D1 базу
wrangler d1 create taskbot_db
# Скопируй database_id в wrangler.toml

# 3. Примени схему
wrangler d1 execute taskbot_db --file=schema.sql --remote

# 4. Добавь токен бота
wrangler secret put BOT_TOKEN

# 5. Деплой
npm install
wrangler deploy

# 6. Настрой webhook
# Открой в браузере: https://твой-воркер.workers.dev/setup
```

## 💰 Стоимость

Полностью бесплатно до 100 000 запросов в день. Для команды до 100 человек — с запасом.

| | Cloudflare Workers | VPS |
|---|---|---|
| Цена | Бесплатно | $5–10/мес |
| Доступность | 99.99% (edge CDN) | ~99% |
| Масштабирование | Автоматическое | Ручное |
| Деплой | `git push` | SSH + скрипты |
| Миграция | Просто код | Настройка сервера |

## 🔧 Полезные команды

wrangler tail

curl https://api.telegram.org/bot<TOKEN>/getWebhookInfo
